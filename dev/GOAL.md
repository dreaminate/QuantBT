## 0. 北极星

面向 **A股 + 加密**、Agent 驱动、**可上线交付**的全栈量化软件，并被一条**不可绕过的机构级治理脊柱**贯穿——让从小白到不懂代码的经济学者，靠对话走完**最严谨、资产无关、中低频**的策略**研究→生产全生命周期**；**人只出意图与经济判断，agent 出全部工程；流程即信任。**

- **资产范围（硬）**：A股 = 研究+回测+paper（**永不实盘**）；加密 = 研究+回测+testnet+**Binance 实盘小额**。中低频，不碰 HFT/做市/延迟套利。明确不在范围：美股/港股/外汇/期货/期权/债券/A股券商网关/其它交易所。
- **目标人群**：小白 ～ 不懂代码的经济学者。
- **唯一原硬约束**：`RunDetailPage` 冻结（只允许排版/显示逻辑/加字段）。
- **"可上线"= 能装 + 能用(10min) + 能造(对话生成策略) + 能信(每run PBO/DSR/Bootstrap) + 能演进(A股到paper/加密到实盘) + 能交付(开放格式可审计)。**

## 1. 终态全景 = 两层相乘

- **层1 · 功能平台**（M1–M21，§7 + 原始 §13 **大部分已建**）：数据→因子→模型→信号→组合→执行→回测→归因→监控 + Agent + IDE + 社区 + 跟单 + 教学 + 实盘安全阶梯。
- **层2 · 机构级 Agent OS 治理**（本阶段在建）：把"零件箱"装配成**不可绕过的治理脊柱**（§2）+ **三纯因子库**（§3）+ **方法学纵深**（§4）+ **上线闭环**（§5）+ **恰当依赖信任层**（§6）。
- **终态 = 层1 的每个能力都被层2 贯穿治理、可证伪验收。** 价值不在再造零件，在接线成不可绕过的闸门。

## 2. 治理脊柱（确定性脊柱 + 受控 LLM 触手 + 治理漏斗）

**终态**：漏斗 `假设→数据→因子→模型→信号→组合→独立验证→审批→上线→监控→退役` 固化为**确定性 DAG**；**LLM 永在节点内、绝不当控制器**（R11）；durable checkpoint / replay（读已落盘工件、不重跑 LLM）/ fork / rollback，**交易副作用设不可幂等边界**（HALT 截断走对账，不重发单）；PROV 谱系总线（谱系≠信任）；**honest-N 一本账**（content-addressed，memoize 与计数同一本，SQLite WAL+JSONL）；可证伪**假设卡预注册**（P2 不挡探索，仅晋级 confirmatory 才冻结）；HITL **双通道审批**（探索轻/确证重可挂数天）；**异模型验证官**（R7 真分离，诚实"非组织独立"）；安全 **deny-by-default + 密钥不进 LLM + 交易所侧硬墙**。
**决策**：R1/R2/R5/R7/R8/R9/R11/R12 + S1–S4。
**可证伪验收**：种泄露→门必抓；种重复 config→不重跑且计 N；N 不可手动改小；approver≠creator；注入也取不到 key；四下单路径同判（M17）；裁决只说"证据充分/不足 + 适用域 + 没验证的"，**绝不说"可信/安全"**。

## 3. 因子轨（三纯库 + 信号契约 + 机构生命周期）

**终态**：**算术（暴力遍历挖掘）/ ML / DL 三库保持纯净**；DL/ML 本体进**模型注册表**、输出登记为**"信号"**进因子库（R17，两层 + 一份信号契约解耦，锚 Qlib/MLflow/Feast）；排列组合/集成**上移到信号层 + 策略层**（防 stacking 泄露：OOF + purge + embargo）；暴力遍历 = **诚实-N 守门人**（生成器/守门器严格解耦，守门指标绝不进生成 fitness）；**机构级因子生命周期**（衰减/拥挤/容量/因子族/退役，跨策略复用的独立资产）；**新闻/事件/链上/A股基本面信号**（中低频）。
**决策**：R13/R15/R16/R17/R18/R19/R21/R22；R20（LLM 引导生成暂缓）。
**验收**：把 `.pt` 当因子塞库=拒（范畴错误）；换等价公式 N_eff 必抓；加密拥挤数据不足→只定性警示、禁自动减仓。

## 4. 方法学纵深（过拟合 / 多重检验 / CV / 不确定性 / 成本 / regime）

**终态**：**多证据三角**（DSR-FST 修正 / PBO-CSCV / bootstrap CI）同向才放行 + **通缩区间**，绝不单点红线（R2）；honest-N（**t>3 不硬编**、三档预注册可切换不锁；Chen 弱识别）；CPCV "更强默认" **双轨** walk-forward（R4）；conformal/CQR/ACI 区间 + abstain（R23）；成本/TCA（平方根冲击 **δ=0.5 窄带** + funding/borrow/印花税，R18）；regime 检测当**风险情景非预测**。
**决策**：R1–R5/R16/R23。
**验收**：噪声策略→PBO≈1/DSR 判过拟合；守门器自身模型风险对用户明示（R5）；DSR=标度修正非修复低估、只与诚实 N 一样诚实；短样本→判"证据不足"非虚假红绿。

## 5. 上线闭环 + 资产无关

**终态**：**live ladder**（backtest→A股paper/加密testnet→小额 live→加码，不可跳级，**M20 SafeKey/Ladder 已建**）；**漂移监控**（rolling-PSR/CUSUM/Page-Hinkley/PSI，**绩效轴主告警 / 特征漂移仅根因**，绝不把 DSR 搬实盘单策略）；kill switch 分级；衰减→降级/退役→问责；**InstrumentSpec / exchange_calendars 声明式资产接入**；**A股纯多头弱腿**（做空不可行，R13）+ 平稳假设本地化（R14）；机构模型治理**改锚 NIST AI RMF**（非 SR11-7，R6，不宣称合规）。
**决策**：R6/R13/R14/R15/R16。

## 6. 信任层（恰当依赖 + 渐进披露 + 反谄媚 + 冷启动 + 个性化）

**终态**：北极星=**恰当依赖**非信任最大化（R24，live 用代理指标）；**L1–L4 渐进披露**——全证据对所有人可下钻、**闸门对所有人一套最严**、默认分层只为可读、**弱点风险一律一等呈现绝不淡化**（R25）；反谄媚**发版门禁**（多轮施压压力测试 + 专家否决权，不看满意度；**专业知识优先于用户 wishful thinking**、可 override、定性非投资建议，R26）；冷启动 N=1 诚实"证据不足"（剔除 DSR 用 PSR/MinTRL + 隐性 champion，R27）；个性化**只动呈现层、不动治理**。
**决策**：R24/R25/R26/R27。
**验收**：红绿灯必一键下钻暴露 有效N/试验聚类/适用域空洞；agent 对"帮我择时稳赢"给现实不迎合；冷启动首日数字标"先验断言、未经数据检验"。
