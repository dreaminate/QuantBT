## 用户验收 3 路径

1. **从注册到第一个证据一致的 run**:
   - 注册 → HomePage 顶部 5 步 onboarding
   - 点 /runs → 跑 demo run → 看 RunDetail 顶部 chip（证据一致/存疑/高风险）
   - 点 Sharpe 旁 ⓘ → 弹 popover L1/L2 → 点"打开专页"跳 /glossary/sharpe_ratio

2. **Mode 2 诊断对话**:
   - /chat → 新对话（绑 active_run_id）
   - 问"PBO 0.7 这个策略证据够吗" → 回复带 RAG hit chip 跳 /glossary/pbo
   - thread 历史持久化

3. **Binance 安全阶梯**:
   - 跑 `/api/trading/safety/ladder` → 起始 level_0
   - SafeKey wizard → 全绿 → can_promote=true → 升 level_1
   - testnet matrix 12/12 → 升 level_2
