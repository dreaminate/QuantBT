## 新增功能（用户能直接看到）

### 1. Glossary 词典基建
- `docs/glossary/` 目录 + 30 条 baseline 术语索引 (`_index.yaml`)
- 3 条完整 markdown 词条（sharpe_ratio / pbo / deflated_sharpe）
- 剩 27 条 `.md` 由用户用 GPT Pro 按 [_PROMPT_FOR_GPT_PRO.md](../glossary/_PROMPT_FOR_GPT_PRO.md) 生成
- 校验脚本 `scripts/validate_glossary.py docs/glossary --min-count 30 --strict-related`

### 2. Glossary REST API
- `GET /api/glossary` — list 全量 summary（支持 `?category=metric&level=advanced` 过滤）
- `GET /api/glossary/{term}` — 通过 slug / alias 拿单条
  - `?level=l1|l2|l3|l4` 渐进披露
  - 404 → `{error, term, suggestions[5]}` (difflib 拼写相近建议)
- `GET /api/glossary_meta` — 词典统计 + related 闭环就绪状态

### 3. RunDetail 字段 ⓘ
- 收益概述每个 metric 旁边一个轻量 ⓘ 按钮
- 点击 → popover 显示 L1 (一句话) + L2 (公式 + 例子)
- "查看 L3 / L4 ↓" 按需展开误区表 + 延伸阅读
- 当前可点击命中: sharpe / sortino / IR / alpha / beta / max_drawdown / volatility 等 7+ 指标
- 严格隔离: 不影响 jq-* 冻结布局，无 tab 删除

### 4. RunDetail 顶部证据状态卡片
- "收益概述" 标题旁加 chip: 证据一致 / 存疑 / 高风险 / 信息不足 (4 档)
- 点击 popover 显示所有 risk_flags 详情 + checked_metrics 列表
- 法律免责 "本卡片仅辅助判断回测证据状态，**不构成投资建议**"
