## Scope

定义量化研究平台第一条实现主线的“研究基座契约”：跨市场数据输入、特征输出、标准决策接口、优化器输入输出、风险约束、回测假设和 artifact 导出边界。

本任务不局限于某一个具体市场。基座必须能支持 A 股、港股、美股和 Crypto；Crypto perpetual 只是高要求适配场景之一，用来验证 funding、杠杆、强平距离等特殊约束不会污染股票市场的通用契约。

本任务不直接实现复杂模型训练，不做实盘交易执行，不扩展前端页面。
